HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The doctype declaration tells the browser what document type it should expect when it is opened.
  • The htlm opening tag represents the root or the start of the html document.
  • Inside the html element there is the attribute lang which tells the browser the language of the website.
  • An html attribute defines additional information about an element.
  • Html attributes are always defined at the begining of the element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When using external css style sheet we the file by using a link element placed in html's head.
  • A selector defines the element or attributes to which the rules or declarations will apply.
  • An element selector add styles to all these elements on the webpage.
  • A class selector allows us to share a css rule to any element we choose.
  • Any html element assigned to the class selector will gain the property assignments from the css.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • git add -A: adds the changes in the current working branch to the staging area.
  • git pull origin main: sync the local branch with the base branch in GitHub.
  • git push origin branch name: to push our changes to the remote branch.

JavaScript

  • A variable is named container that allows us to store data in our code.
  • Control flow is the order in which the computer executes code in a script.
  • An If statement is a way of introducing decision-making into our code.
  • An array is a single variable that is used to hold a group of data.
  • A function is a set of instructions that tells the computer how to perform a certain task.
